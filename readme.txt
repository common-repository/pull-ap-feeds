== Installation ==

1. Upload `pull-ap-feeds` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure by clicking on "AP feed settings" in the "Settings" submenu on the left-side of wp-admin.
